# eusoujulia.com.br

Site pessoal de **Júlia Alberto**, com foco em **Design Ops**, **Design Systems** e **Acessibilidade**.

Este projeto foi pensado para ser:

- simples
- sem dependências desnecessárias
- semanticamente correto
- acessível
- fácil de manter

Sem frameworks, sem build complexo, sem overengineering.

---

## ✨ Objetivos do projeto

- Servir como site pessoal e portfólio
- Experimentar boas práticas de HTML, CSS e JS “vanilla”
- Priorizar acessibilidade desde a base
- Manter uma arquitetura simples e compreensível
- Ter suporte a **tema claro/escuro** e **idiomas (PT/EN)**

---

## 🧱 Stack

- **HTML5 semântico**
- **SCSS** (organizado por responsabilidade)
- **JavaScript vanilla**
- Nenhum framework
- Nenhuma lib de i18n
- Nenhum bundler
